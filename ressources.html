Doc Bootstrap Cards 		J'ai commencé le projet par faire un template de ma page puis la page d'accueil donc une partie des sources citées dans cette ligne sont utilisées pour toutes les pages.
Dans la partie hamburger ainsi que dans la partie cards, je me suis principalement aidé de la documentation sauf que pour debug quelques micro-fonctionnalités, j'ai utilisé Gemini.
De plus, lorsque j'ai voulu changer les couleurs par défaut de bootstrap, j'ai utilisé Gemini principalement pour trouver des couleurs qui allaient bien par rapport à mes goûts et ce qui me représente. Je n'ai utilisé que le css pour modifier les couleurs des variables et classes bootstrap natives.
